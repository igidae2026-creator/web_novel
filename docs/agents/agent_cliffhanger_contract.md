# Agent Cliffhanger Proposal

Replace keyword detection with a real cliffhanger generator.

## Responsibilities

- derive cliffhanger type from active event and unresolved threads
- produce withheld consequence language
- validate next-episode pull strength

## Cliffhanger Modes

- revelation-cut
- choice-lock
- threat-imminent
- reversal-afterglow
- sacrifice-fallout
- arrival-shock

## Success Criteria

- points to a concrete open question
- points to a concrete consequence
- increases unresolved pressure in retention state

## Failure To Avoid

- punctuation-only suspense
- generic “but then” endings
