# Agent Cliffhanger Tension

## Diagnosis

Cliffhanger and tension systems existed but were not tied enough to irreversible pressure and serialization sustainability.

## Ceiling Bottleneck

Generic suspense increases clicks only briefly and burns trust.

## Missing Architecture

- irreversible cost in closing line
- sustainability-aware intensity modulation
- rhythm window memory

## Proposed Subsystem

Use event type + conflict consequence + pacing debt to choose cut mode and produce withheld consequence with a real future cost.

## Integration Points

- retention engine
- analytics ceiling evaluator

## Regression Risks

- every chapter ending on max spike
- empty question-mark suspense

## Test Plan

- weak cliffhanger gets replaced
- high-pressure states select spike band
- repeated peaks accumulate release debt
