# Agent Retention Proposal

Upgrade retention using unresolved thread pressure.

## Responsibilities

- estimate click-through pressure into the next episode
- score unresolved threads by urgency and cost
- combine cliffhanger carryover with payoff debt
- identify weak continuation hooks

## Required Signals

- unresolved thread count
- threat proximity
- promised payoff debt
- curiosity debt
- recent loss or betrayal carryover

## Failure To Avoid

- retention predicted only from quality scores
- ignoring unresolved consequence pressure
