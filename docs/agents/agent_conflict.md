# Agent Conflict

## Diagnosis

Conflict escalation relied too much on generic pressure counts.

## Ceiling Bottleneck

Escalation without irreversible consequences yields hollow intensity.

## Missing Architecture

- irreversible_if_lost
- risk distribution across physical/social/moral/strategic
- world-side fallout

## Proposed Subsystem

Upgrade conflict threads to carry explicit irreversible costs and risk distribution, then propagate fallout into world state.

## Integration Points

- typed event selection
- cliffhanger withheld consequence
- regression guard via world logic and sustainability

## Regression Risks

- too many open threads
- escalation can outpace payoff

## Test Plan

- loss creates fallout thread
- high consequence state changes escalation mode
- world instability rises after heavy fallout
